# World Cup 2026 Guide

A React-based interactive FIFA World Cup 2026 guide website.

## Project Introduction

This project is an interactive web application for introducing the FIFA World Cup 2026.

Main features include:

* World Cup countdown timer
* World Cup history and champions
* Basic football rules and offside diagram
* National team profiles and squad information
* Match schedule and match detail pages
* Favorite teams and matches
* Light / Dark Theme
* English / Chinese (i18n) language switching
* Responsive Web Design (RWD)

## Technologies Used

* React
* Vite
* React Router
* React Hooks (`useState`, `useEffect`, `useMemo`, `useCallback`)
* Context API
* Custom Hooks
* i18next (i18n)
* localStorage
* CSS / Responsive Design

## Installation & Run

Clone this repository:

```bash
git clone https://github.com/The-Pancake-man/World-Cup-2026-Guide.git
```

Install dependencies:

```bash
pnpm install
```

Run development server:

```bash
pnpm dev
```

Build project:

```bash
pnpm build
```
